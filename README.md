# To-Do List — HTML / CSS / JavaScript

**Лёгкое приложение для управления задачами на чистом JavaScript — без сборщиков и зависимостей, просто открыть и пользоваться.**

## ✨ Возможности

- Добавление задач с текстом и временем выполнения
- Отметка задач выполненными (клик по кружку-статусу)
- Редактирование задачи прямо в списке (без `prompt`)
- Удаление отдельных задач с анимацией
- Фильтры: **Все** / **Активные** / **Готово**
- Счётчик задач и отдельный счётчик выполненных
- Кнопка «Очистить выполненные»
- Автосортировка задач по времени
- Сохранение в `localStorage` — список не пропадает при перезагрузке страницы
- Пустое состояние с подсказкой, когда список пуст
- Адаптивная вёрстка и тёмная тема с акцентным цветом
- Видимый фокус для клавиатурной навигации и поддержка `prefers-reduced-motion`

## 🛠 Технологии

- **HTML5** — семантическая структура, `<template>` для карточек задач
- **CSS3** — кастомные свойства (переменные), флексбокс, анимации, тёмная палитра
- **Vanilla JavaScript (ES6+)** — без фреймворков и сборки
- **Web Storage API** — хранение задач в `localStorage`
- Шрифты **Manrope** и **JetBrains Mono** через Google Fonts

## 🚀 Как запустить проект

Проект не требует установки зависимостей или сборки — это обычные статические файлы.

**Вариант 1 — просто открыть файл:**

Откройте `index.html` в браузере двойным щелчком.

**Вариант 2 — через локальный сервер (рекомендуется):**

# Клонировать репозиторий
git clone https://github.com/Topchikdan/to-do-list-vue-project-additions.git

# Перейти в папку проекта
cd to-do-list-vue-project-additions

# Запустить локальный сервер, например через Python
python3 -m http.server 8000

Затем откройте `http://localhost:8000` в браузере.
